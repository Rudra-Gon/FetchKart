FetchKart — E-Commerce Platform
A barebones e-commerce website built with pure PHP, HTML, and CSS featuring a glassmorphism UI.
Features

Product listing with a clean browse layout
Cart system with PHP session-based state tracking
Cart management — view totals, quantities, and remove items
Simulated checkout flow

Requirements

PHP 7.x or 8.x
XAMPP, WAMP, or any local Apache server

Setup

Copy the project folder into your htdocs directory
Start Apache from the XAMPP Control Panel
Visit http://localhost/Microtech_internship_ECOM/index.php
